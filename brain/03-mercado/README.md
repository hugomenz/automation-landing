# Mercado

## Objetivo

Medir si puedes conseguir oportunidades que superen tu situación actual.

## Lectura rápida

El frontend employee normal no necesariamente mejora tu situación. El freelance/contractor, internal tools, automation y roles de arquitectura sí tienen más upside.
