# Leads

Tener una lista accionable para contactar empresas, aplicar a ofertas y hablar con recruiters esta semana.

3. Agencias que subcontratan desarrollo.
4. Recruiters con proyectos contract/freelance.
5. Plataformas con proyectos directos.
