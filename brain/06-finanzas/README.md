# Finanzas

## Objetivo

Calcular de forma prudente qué tendrías que facturar para llegar a 4.800 € netos/mes como autónomo en Alemania.

## Aviso

Esto es una estimación práctica, no asesoría fiscal. Para exactitud usa Steuerberater y calculadoras oficiales.
