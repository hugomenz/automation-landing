# Escenarios

## Conservador

- Facturación: 110.000 €/año.
- Días facturables: 160–170.
- Tarifa: 650–700 €/día.
- Mejor protección ante gaps.
- Usaría este escenario para decidir salida.

## Realista

- Facturación: 105.000 €/año.
- Días facturables: 175–185.
- Tarifa: 580–650 €/día.
- Viable con 1 cliente principal + algunos sprints pequeños.

## Optimista

- Facturación: 95.000–100.000 €/año.
- Alta ocupación.
- Pocos gaps.
- Menores gastos.
- No usaría este escenario como base para dejar empleo.

## Regla de decisión

Si tras 6–8 semanas no ves una ruta creíble hacia 580–700 €/día o 80–95 €/h, mantendría el empleo y seguiría validando a tiempo parcial.
