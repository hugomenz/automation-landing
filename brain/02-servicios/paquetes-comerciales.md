# Paquetes comerciales

| Paquete | Qué incluye | Duración | Precio sugerido de entrada |
|---|---|---:|---:|
| Bottleneck Audit | discovery, mapa de proceso, quick wins, propuesta | 3–5 días | 900–1.800 € |
| Automation Sprint | 1 workflow real + documentación + handover | 1–2 semanas | 2.500–5.500 € |
| Internal Tool MVP | UI interna + API + flujo operativo | 2–4 semanas | 5.500–12.000 € |
| Angular Stabilization Sprint | auditoría, fixes críticos, arquitectura y plan | 1–2 semanas | 2.500–6.000 € |
| Design System Kickstart | componentes base, Storybook, tokens, guidelines | 1–3 semanas | 3.500–9.000 € |
| Copilot Adoption Sprint | workshop, playbook, guardrails y ejemplos | 1 semana | 1.500–3.500 € |

## Cómo venderlos

No empieces con “puedo trabajar por hora”. Empieza con un paquete pequeño.

Ejemplo:

> Puedo empezar con un audit corto. Si vemos ROI claro, lo convertimos en un sprint de implementación.
