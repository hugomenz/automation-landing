# Servicios principales

## 1. Internal tools y portales operativos

Para empresas que tienen procesos internos lentos o herramientas improvisadas.

### Entregables

- Dashboard interno.
- CRUD operativo.
- Integración API.
- Auth básica.
- Documentación de uso.
- Handover técnico.

## 2. Angular rescue / frontend architecture

Para equipos con deuda frontend, Angular difícil de mantener o entregas lentas.

### Entregables

- Auditoría de arquitectura.
- Refactor de módulos críticos.
- Componentes base.
- Estándares de testing.
- Roadmap de mejora.

## 3. Design systems aplicados

Para equipos que necesitan consistencia UI y velocidad de desarrollo.

### Entregables

- Component library.
- Storybook.
- Design tokens.
- Guidelines.
- Ejemplos reales de uso.

## 4. Workflow automation

Para operaciones, soporte, ventas, finanzas o HR.

### Entregables

- Workflow n8n/Make/Power Automate.
- Integraciones API.
- Logging.
- Fallbacks.
- Documentación.

## 5. Copilot / AI-assisted development rollout

Para equipos que quieren usar IA sin bajar calidad.

### Entregables

- Playbook.
- Guardrails.
- Prompt examples.
- PR checklist.
- Workshop.
- Métricas de adopción.
