# Portfolio

## Objetivo

Crear un portfolio que venda servicios, no solo skills técnicas.

Cada pieza debe dejar claro:

- qué problema de negocio resuelve,
- qué entregable produces,
- qué resultado podría medir el cliente,
- por qué tu experiencia parece senior,
- qué siguiente paso debe tomar el comprador.

## Regla de portfolio

No enseñes proyectos como “mira lo que programé”. Enséñalos como “este tipo de problema sé convertirlo en un sistema mantenible”.
