# Portfolio mínimo 48h

## Objetivo

Tener algo suficientemente serio para hacer outreach esta semana.

## Entregables en 48 horas

- [ ] Landing simple con propuesta de valor.
- [ ] 3 servicios claros.
- [ ] 3 mini casos de estudio.
- [ ] 3 READMEs profesionales en GitHub.
- [ ] PDF comercial de 2 páginas.
- [ ] CTA para llamada de 15 minutos.
- [ ] Página específica para cold outreach.
- [ ] Página específica para plataformas freelance.

## Landing mínima

### Hero

> Ayudo a equipos a reducir trabajo manual y acelerar productos internos con Angular, design systems, APIs y automatización.

### Dolores

- procesos internos lentos,
- herramientas internas difíciles de usar,
- frontend Angular con deuda técnica,
- design system inexistente o mal adoptado,
- Copilot/IA usado sin método,
- datos movidos manualmente entre herramientas.

### Servicios

1. Internal tools y automatización.
2. Angular/frontend architecture y design systems.
3. GitHub Copilot / AI-assisted development rollout.

### CTA

> Si tienes un cuello de botella claro, te digo en 15 minutos si tiene sentido resolverlo conmigo.

## Qué no incluir

- Blog.
- Historia personal larga.
- Lista enorme de tecnologías.
- Proyectos sin resultado de negocio.
- Claims como “aumento productividad 10x” sin evidencia.

## Stack rápido

| Opción | Cuándo usarla |
|---|---|
| Astro | landing simple y rápida |
| Next.js | si quieres algo más productizado |
| Notion público | si necesitas publicar hoy |
| GitHub Pages | si quieres coste cero |
| PDF + LinkedIn | si todavía no tienes web |
